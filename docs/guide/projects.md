# 项目管理

项目（Project）用于在组织下对应用进行分组和权限隔离。

## 层级关系

```
组织 (Organization)
  └── 项目 (Project)
        └── 应用 (Application)
```

- **系统管理员**：可管理所有组织与项目。
- **组织管理员**：可管理本组织下的项目及应用。
- **项目管理员**：可管理本项目下的应用与权限。

## 创建项目

1. 进入 **组织管理**，选择目标组织。
2. 在该组织下进入 **项目管理**，点击 **新建项目**。
3. 填写项目名称、描述等，保存。

## 权限说明

- 项目与组织、用户角色配合，实现多租户与 RBAC。
- 用户仅能访问其所属组织/项目下的应用与资源。
