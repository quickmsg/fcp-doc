# 用户与权限

FCP 采用基于角色的访问控制（RBAC），支持多级角色。

## 用户角色

| 角色 | 说明 |
|------|------|
| **系统管理员 (SystemAdmin)** | 管理所有组织、用户、审计日志，最高权限。 |
| **组织管理员 (OrganizationAdmin)** | 管理本组织下的项目、用户、应用及本组织告警与审计。 |
| **项目管理员 (ProjectAdmin)** | 管理本项目下的应用与相关告警。 |
| **普通用户 (User)** | 仅可查看有权限的项目与应用，权限最小。 |

## 登录与认证

- 登录接口：`POST /api/v1/auth/login`，成功后返回 JWT。
- 后续请求在 Header 中携带：`Authorization: Bearer <token>`。
- Token 有效期由配置 `jwt.expiration` 控制。

## 用户管理

- **系统管理员**：可创建/编辑/禁用用户，并分配所属组织与角色。
- **组织管理员**：可管理本组织下的用户与角色。
- 首次部署时若开启 `admin.auto_create`，将自动创建配置中的管理员账号。
