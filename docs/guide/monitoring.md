# 监控

FCP 提供对已纳管 FluxMQ 集群的监控能力。

## 监控内容

- **集群健康**：节点在线状态、连接数、消息吞吐等（依赖 FluxMQ 上报或 FCP 拉取）。
- **系统状态**：FCP 自身资源与健康（若实现）。

## 使用方式

1. 在 **应用** 或 **监控** 页面选择目标应用/集群。
2. 查看概览指标：连接数、消息 TPS、延迟等。
3. 可为应用配置**监控采集间隔**（如 60 秒），FCP 按间隔拉取或接收指标。

## 与告警的关系

- 监控数据可作为**告警规则**的数据源。
- 在 [告警管理](alerts.md) 中配置规则后，异常时触发告警并发送到通知渠道。
